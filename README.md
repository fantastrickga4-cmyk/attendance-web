# 직원 출퇴근 관리 (웹 버전)

브라우저에서 동작하는 간단한 직원 출퇴근 관리 시스템이에요.

## ✨ 기능
- 🔐 회원가입 / 로그인 (비밀번호 SHA-256 해시 저장)
- 🟢 출근 찍기 / 🔴 퇴근 찍기 (하루 한 번)
- 📋 내 출퇴근 기록 보기 (근무 시간 자동 계산)
- 💾 데이터는 브라우저 localStorage에 저장

## 🚀 사용 방법

### 로컬에서 실행
1. 이 폴더의 `index.html`을 더블클릭하면 끝!
2. (또는) Node.js 설치되어 있다면 `npx serve` 같은 정적 서버로 실행

### 배포된 사이트
Vercel에 배포되어 있어요 → 자세한 URL은 GitHub README 또는 Vercel 대시보드에서 확인

## 📁 파일 구성
- `index.html` — 화면 구조 (HTML)
- `style.css` — 디자인 (CSS)
- `app.js` — 동작 (JavaScript)

## ⚠️ 주의
- 데이터는 **각 브라우저에만** 저장돼요. 다른 사람과는 공유되지 않아요.
- 진짜 회사처럼 모두가 같은 데이터를 보려면 데이터베이스(Vercel KV/Postgres 등) 추가가 필요해요.
