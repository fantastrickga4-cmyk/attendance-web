🔮 신당 키우기

한국 무속 세계관 기반 콘텐츠 깊이형 Incremental RPG

장르 Incremental / Idle RPG 플랫폼 Android (Play Store) 엔진 Unity 6 LTS 개발 기간 8~10주 (1인) 목표 r/incremental_games 바이럴 → IAP 수익화

게임 컨셉

작은 신당을 운영하는 무당. 굿을 올려 신령의 신뢰를 쌓고, 의뢰인의 소원을 들어주며 복채를 모은다.
신뢰가 높아질수록 더 강한 신령이 강림하고, 더 큰 소원을 이뤄줄 수 있게 된다.
오방신장 → 산신 → 용왕 → 옥황상제까지 한국 신화 pantheon을 트리 구조로 모두 해금하는 것이 최종 목표.

메타 루프 (세션 간)
앱을 꺼도 오프라인 굿판이 자동 진행.
재접속 시 "신령이 XX건을 처리했습니다" 연출.

복채 축적 → 신당 확장 → 더 많은 신령 슬롯 → 더 빠른 복채 생산.
수직 성장이 핵심 재미.
프레스티지 시스템
다음 생(轉生): 모든 것을 리셋하고 영구 보너스 획득.

리셋 보상: 무당 레벨 ×1.5 영구 복채 배율
전생 횟수에 따라 새 신령 계보 해금.
"더 빠르게 처음부터" 중독성 유발.

자원 & 시스템

💰 복채 (골드)
기본 화폐. 의뢰 완료 시 획득.
신당 업그레이드, 제물 구매에 사용.
오프라인에도 쌓임 (최대 8시간).
✨ 신기(神氣)
프리미엄 화폐 (IAP 또는 보상광고 획득).
신령 즉시 소환, 슬롯 확장, 속도 부스터에 사용.
광고 보면 소량 획득 가능.
🌟 신뢰도
신령별 개별 수치.
굿 횟수 누적 시 상승.
임계값 도달 → 신령 등급 승급.
숫자가 커지는 핵심 지표.

의뢰 시스템

의뢰 종류
일반 의뢰 — 자동 처리, 낮은 복채
(예: 합격 기원, 건강 회복, 연애 성취)

특별 의뢰 — 탭 인터랙션 필요, 높은 복채
(예: 조상 천도, 집안 액막이, 사업 번창)

긴급 의뢰 — 한정 시간, 완료 시 신기 보상
(예: 출산 임박, 큰 수술 전날)
계절 이벤트
동짓날 — 팥죽 제물 이벤트, 복채 2배
추석 — 조상 굿판, 희귀 신령 출현율↑
정월 대보름 — 소원 처리 슬롯 +2 (24시간)
음력 생일 — 무당 레벨업 보너스

실제 한국 절기 날짜 기반으로 자동 트리거.

Unity 씬 & 컴포넌트 구조

씬 구성
  • Boot — 저장 데이터 로드, 초기화
  • MainGame — 신당 메인 화면
  • ShrineMap — 신령 트리 / 해금 화면
  • Shop — IAP / 신기 상점
  • Ritual — 굿 진행 팝업
  • Prestige — 전생 확인 화면
핵심 매니저 (싱글톤)
  • GameManager — 전체 상태 관리
  • SaveManager — JSON 직렬화 로컬 저장
  • OfflineManager — 오프라인 경과 계산
  • DeityManager — 신령 해금/레벨 관리
  • QuestManager — 의뢰 큐 처리
  • IAPManager — Unity IAP 래퍼
  • AdsManager — AdMob 보상광고 래퍼
데이터 구조 (ScriptableObject)
  • DeityData — 신령 스탯, 보너스, 아이콘
  • QuestData — 의뢰 종류, 복채, 소요시간
  • UpgradeData — 업그레이드 비용 곡선
  • SeasonEventData — 이벤트 날짜, 보상
수식 설계 (밸런스)
  • 복채 생산: base × tier^1.8 × bonuses
  • 업그레이드 비용: base × 1.15^level
  • 신뢰도 요구: 100 × 2^(tier-1)
  • 오프라인 효율: 온라인 대비 60%
  • 전생 보너스: +5% 복채 배율/회
