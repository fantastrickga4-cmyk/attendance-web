02 핵심 수치 시스템 (Unity 코드로 바로 사용)

💰 복채 생산 공식
// 전체 BK/s BK/s = Σ (base × LevelMult(lv) × SpiritMult × RegionMult × EraMult) // 레벨 배율 LevelMult(lv) = lv^1.5 × 1.2^(lv / 25) // 업그레이드 비용 Cost(lv) = base × 1.15^lv × Compress(lv) // Compress: lv<25→1.0, lv<50→0.85, lv<100→0.70
🌙 오프라인 수익 공식
offline_BK = BK/s × min(elapsed, cap) × efficiency × (1 + shamanLv × 0.002) efficiency: 기본: 0.25 (25%) 자동화 제단 구매: 0.50 야광귀 신령: 0.60 프리미엄 굿 완료: 0.75 (24h) cap (시간 상한): 기본: 14,400s (4h) 수호 신령: 28,800s (8h) 3회 전생+: 43,200s (12h)
⚡ 신뢰도 요구량
TR_required(tier) = 100 × 2.8^(tier - 1) tier=1: 100 (~10분) tier=2: 280 (~30분) tier=3: 784 (~2시간) tier=4: 2,195 (~6시간) tier=5: 6,147 (~15시간) tier=6: 17,211 (~30시간) tier=7: 48,191 (~60시간) tier=8: 134,934 (100h+) TR/s = (BK/s × 0.01) + 굿완료 보너스 소굿: +50 중굿: +200 대굿: +1,000
🔄 프레스티지 SK 공식
SK_gain = floor(sqrt(totalBK / 1,000,000)) × eraMult eraMult: 현대(1회): ×1.0 조선(2회): ×1.3 고려(3회): ×1.7 고조선(4회): ×2.2 신계(5회): ×3.0 영구 복채 배율: SK 0~500: 각 SK = +1% (상한 +500%) SK 500~2K: 각 SK = +0.5% SK 2K~10K: 각 SK = +0.1% SK 10K+: BK배율 고정, 신계 콘텐츠 화폐 전환

03 레이어 해금 타이밍 & 리텐션 설계

⏱️ 해금 타이밍 (이탈 방지선)
0:00 ~ 5:00
D1 훅 — 삼신할미 자동 강림
3분 내 자동 발동. 플레이어가 아무것도 안 해도 첫 신령 강림 연출 발생. 이탈 최대 위험구간.
0:45 ~ 1:00
Layer 2 진입 — 무당 고용
45분 미만: 루프 파괴. 90분 초과: 이탈(ChurnPoint #1). 반드시 이 창에 진입 유도.
6 ~ 8시간
Layer 3 진입 — 지역 확장
D1 종료 전 반드시 진입. 지도 UI 등장 + 지역 조명 켜지는 연출 → "와" 모멘트 #2.
15 ~ 20시간
Layer 4 진입 — 제자 시스템
D2 세션에서 경험. 자동화 레이어 추가 → "게임이 커진다" 느낌.
35 ~ 50시간
첫 프레스티지 권유
조건: 누적BK ≥ 1B, L4 해금, 신령 10종+, 플레이 35h+. SK 예상량 HUD 표시 → "전생 욕구" 극대화.
30 ~ 40시간
Layer 5 — 신계 탐험 (v1.0 초안)
v1.0은 "파견→대기→귀환" 타이머 방식으로 단순 구현. 풀 콘텐츠는 v1.1.
